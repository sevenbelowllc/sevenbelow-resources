# LangChain Homework 2026

DataExpert.io 2026 AI Engineering Bootcamp — LangChain / LangGraph module.

## Summary

Three assignments covering agent design, context management, and middleware patterns:

1. **Context overflow** — managing long-running conversation state
2. **Infinite researcher** — bounded research-agent loop
3. **MCP middleware** — Model Context Protocol middleware integration

Each assignment ships with runtime logs, token-usage capture, and a writeup.

## Layout

- `assignment1_context_overflow/`, `assignment2_infinite_researcher/`, `assignment3_mcp_middleware/` — per-assignment work
- `by_assignment/` — token reports broken out per assignment
- `planning/` — project plan
- `token_report.*`, `_run_capture.py`, `token_report.py` — aggregate token-usage reporting tools and outputs

Solutions and assignment briefs are not reproduced here.
