# OpenClaw Week 4 Homework

DataExpert.io 2026 AI Engineering Bootcamp — OpenClaw module, week 4.

## Summary

Debug-and-harden exercise on an OpenClaw agent. Scope covers a fixed agent build, LangSmith-based evaluation, safety/tool/model/classifier tests, before/after metrics capture, and a preflight check.

## Layout

- `solution/` — packaged agent (`openclaw_agent`), evals, tests, metrics, scripts
- `fixed_openclaw_agent.py` — entry runner for the fixed agent
- `preflight.py` — pre-run environment check
- `RUN.md` — runtime setup and execution instructions
- `tests.md`, `fix-report.md`, `SUBMISSION.md` — test notes, fix narrative, submission summary

Solutions and assignment briefs are not reproduced here.
